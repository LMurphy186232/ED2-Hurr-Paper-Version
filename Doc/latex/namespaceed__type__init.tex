\doxysubsection{Detailed Description}
Sub-\/routines that initialise most global variables in ED-\/2. 

These sub-\/routines assign initial values for most global variables in ED-\/2. Every variable must be assigned an initial value before it is used in the right-\/hand side of another assignment, or in any logical test. \begin{DoxyAuthor}{Author}
