\doxysubsection{Detailed Description}
Routines that copy contents from, to or between rk4 structures. 

These subroutines copy the content either from csite to rk4patch or between two rk4patch objects, or from rk4patch objects back to csite. Sub-\/routine copy\+\_\+rk4\+\_\+patch had been moved to a separate module to avoid circularities. ~\newline
 The module was expanded to include the other routines that copy variables, to organise the code.
