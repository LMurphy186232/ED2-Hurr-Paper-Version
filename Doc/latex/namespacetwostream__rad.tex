\doxysubsection{Detailed Description}
Sub-\/routines that solve the two-\/stream model for SW and LW radiation ! 

These sub-\/routines solve the multi-\/layer two stream model to obtain the ! net absorption of irradiance by each cohort. These routines are similar to ! and intended to replace them eventually. ! \begin{DoxyAuthor}{Author}
Feb 2013 First development, based on Liou\textquotesingle{}s book (Marcos Longo) ! 

21 Oct 2017 Turned the file into a module for safer compilation (Marcos Longo) ! 
\end{DoxyAuthor}
