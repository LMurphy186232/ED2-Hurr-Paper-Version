Detailed Description

Sub-routines that handle carbon:nitrogen budgets and sthoichiometry.

This module contains some sub-routines that relate carbon and nitrogen and assess their budgets. These routines used to be in structural growth, however they were migrated to a separate module to avoid circular dependency.
